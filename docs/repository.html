The Mesa 3D Graphics Library

Source Code Repository

The master Git repository is hosted on freedesktop.org with GitLab.

There's also a Mesa Demos and Tests repository.

For a brief overview of the Mesa source tree organization see the (possibly somewhate out of date) Source Code Tree page.

Anonymous Git Cloning

If you simply want to anonymously clone the repository to your local system so that you may build Mesa and use it: 

git clone https://gitlab.freedesktop.org/mesa/mesa.git

Later, you can update your tree from the master repository with: 

git pull origin

And if you want to clone the Mesa demos/tests repository: 

git clone https://gitlab.freedesktop.org/mesa/demos.git

Contributing as a Developer

Contributing to the project involves creating a GitLab account, forking the git repository and contributing patches. See the detailed GitLab instructions and be sure to read the coding style and patch style instructions.
